# Tarea 03 HTML Individual

Esta entrega corresponde a una pieza narrativa individual desarrollada en HTML para integrarse dentro de la webstory grupal **¿Escuchamos lo mismo en Latinoamérica?**

## Tema de la pieza

Se enfoca en el caso de Chile y observa la presencia de artistas chilenos y no chilenos dentro del Top 50 de Spotify. El objetivo es mostrar cómo el ranking chileno combina música local con tendencias latinoamericanas e internacionales.

## Proceso de trabajo

Para desarrollar esta tarea retomé el trabajo realizado en el proyecto grupal y seleccioné una visualización que permitiera contar una historia específica sobre Chile. En vez de enfocarme solo en las canciones más escuchadas, decidí observar quiénes aparecen detrás de esas canciones y qué lugar ocupan los artistas nacionales dentro del ranking. A partir de esa idea organicé la página como una cápsula narrativa, con una introducción, una crónica breve, la visualización, un hallazgo principal y una fuente de datos. El sitio fue construido con HTML, usando una estructura simple y clara para ordenar el contenido y facilitar la lectura. Finalmente, la página fue publicada mediante GitHub Pages para que pudiera revisarse en línea.
