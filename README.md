# Tarea 03 HTML Individual

Esta entrega corresponde a una pieza narrativa individual desarrollada en HTML para integrarse dentro de la webstory grupal **¿Escuchamos lo mismo en Latinoamérica?**

## Tema de la pieza

La cápsula se enfoca en el caso de Chile y observa la presencia de artistas chilenos y no chilenos dentro del Top 50 de Spotify. El objetivo es mostrar cómo el ranking chileno combina música local con tendencias latinoamericanas e internacionales.

## Proceso de trabajo

Para realizar la página, primero organicé el contenido narrativo de la pieza: título, contexto, crónica, visualización, hallazgo principal, ideas clave y fuente. Luego estructuré el archivo `index.html` utilizando etiquetas básicas de HTML revisadas en clases, como `head`, `body`, `header`, `div`, `h1`, `h2`, `p`, `ul`, `li`, `img`, `a` y `footer`.

La visualización fue elaborada a partir de la base oficial del proyecto, comparando la presencia de artistas chilenos y no chilenos en el ranking de Spotify en Chile durante los meses analizados.
